Object-OrientedProgramminglanguage :

	Object-OrientedProgramming(OOP)language is based 
upon the objects.object-oriented programming is about cr	eating objects that contain both data and functions. In this programming language, programs are divided into small parts called objects. It is used to implement real-world entities like inheritance, polymorphism, abstraction, etc in the program to make the program resusable, efficient, and easy-to-use.
